# \[WIP\]: SQS GUI

Graphical user interface to manage SQS instances

## Local development

### Install and run

Using python 3.7+

```powershell
py -3 -m venv venv
./venv/bin/activate.ps
pip install -r requirements-dev.txt

cd local
docker-compose -p sqs-gui up -d

py -3 main.py
```
